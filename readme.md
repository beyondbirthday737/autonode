# Autonode CLI

Uma ferramenta para automatizar a criação de projetos em node.js.


## Instalação
1) Clone esse repositório 
2) Entre na pasta do projeto
3) npm link

## Comandos
* autonode genexp -> Este comando irá gerar uma estrutura de projeto em express.js

## Avisos
